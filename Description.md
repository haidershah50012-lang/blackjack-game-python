# Hangman Game 🎮

A simple Hangman game built with Python where the player guesses 
letters to complete a hidden word.

## How it Works
- A random word is selected from a word list
- The word is displayed as underscores ( _ )
- Player guesses one letter at a time
- Player has 6 lives to guess the word
- If a wrong letter is guessed, a life is lost
- Player wins by completing the word before lives run out

## Features
- Random word selection
- Live tracking system
- Simple terminal based gameplay

## Libraries Used
- random (built-in Python library)

## How to Run
- Make sure Python is installed on your computer
- Download the hangman.py file
- Open terminal and run: python hangman.py
